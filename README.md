# LudoGame (Cờ cá ngựa mobile)
##Mô tả: Trò chơi cờ cá ngựa trên điện thoại có thể chơi được từ 3-5 người. 
- Có 2 ngôn ngữ tiếng anh và tiếng việt
- Chơi trên cùng 1 thiết bị
- có thể chọn số người chơi và máy còn lại
